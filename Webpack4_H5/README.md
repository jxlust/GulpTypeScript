# Webpack4_H5
移动端适配viewport
### webpack4 demo
------
>webpack4 多入口，多页面项目构建案例。
#### npm install
```
npm install //或
cnpm install
```
### run
```
npm run dev //开发环境
npm run build //生产环境打包
```
